# Research Output Modes

Choose the mode that matches the user's requested output. If unclear, select the smallest mode that satisfies the request and state the assumption.

## report_parsing

Use for market updates, protocol reports, research notes, exchange data, news, papers, and structured datasets. Extract key facts, normalize terminology, cite evidence, and separate source facts from interpretation.

## factor_engineering

Use when the user wants candidate factors, screening logic, signal hypotheses, or quant-ready research. Produce factor candidates using the required seven-field contract and mark validation status explicitly.

## philosophical_analysis

Use when the user wants deeper interpretation of assumptions, causal claims, uncertainty, narratives, or failure modes. Apply only the thinking models needed for the question and keep conclusions tied to evidence.
